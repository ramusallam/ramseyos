# Workshop Development Playbook

**Workspace:** Woven
**Workflow ID:** `woven-workshop-development`

## Purpose

Generate structured outlines and session plans for professional development workshops. Workshops should be engaging, practical, and tailored to the specific audience and client context.

## Inputs

- **Workshop Title** — Title or working name.
- **Audience** — Who the workshop is for (e.g., K-12 teachers, school leaders, district administrators).
- **Duration** — Total workshop length.
- **Objectives** — What participants should learn or be able to do.
- **Constraints** (optional) — Client requests, logistical limitations, or scope notes.

## Process

1. Review objectives and audience to determine the right level and approach.
2. Design a session flow with clear segments: opening, core content, activities, and closing.
3. Include interactive elements — workshops should not be lectures.
4. Add timing estimates for each segment.
5. Note required materials or preparation.
6. Output a complete workshop outline.

## Outputs

- A structured workshop outline in markdown with session flow, activities, timing, and materials.

## Notes

- Workshops should reflect Ramsey's facilitation style: interactive, grounded in practice, intellectually engaging.
- Always include participant activity. Pure presentation segments should be short.
- If constraints are provided, the outline must respect them explicitly.
- Review is required before sharing with clients or using in delivery.
